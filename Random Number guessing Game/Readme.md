Number Guessing Game


## Description

This Python project is an interactive Number Guessing Game implemented using the Tkinter library. It allows players to guess a randomly generated number between 1 and 100 within a limited number of attempts. The game provides hints, tracks the number of attempts, and congratulates the player upon successful guesses.

## Features

- User-friendly graphical user interface (GUI).
- Random generation of a secret number for each game.
- Hints provided for each guess (too low or too high).
- User input validation to ensure guesses are within the valid range.
- Option to play the game multiple times or exit.

## How to Play

1. Enter your name in the provided input field.
2. Click the "Play Game" button.
3. You will be welcomed and informed of the game rules.
4. Start guessing the secret number within 10 attempts.
5. Receive hints after each guess.
6. Win by guessing the correct number or lose after 10 attempts.
7. Decide whether to play again or exit.

## Prerequisites

- Python (3.6 or higher)
- Tkinter library (usually included with Python installations)

## Installation

1. Clone the repository to your local machine
2. Navigate to the project directory
3. Run the game
